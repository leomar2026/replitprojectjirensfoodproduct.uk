# Maintenance Checklist

- [ ] Review PM2 logs weekly.
- [ ] Review Nginx logs weekly.
- [ ] Confirm database backups complete daily.
- [ ] Test restore process monthly.
- [ ] Rotate admin passwords regularly.
- [ ] Remove inactive staff accounts.
- [ ] Review Manager and Cashier permissions after role changes.
- [ ] Clean old uploaded receipts/images according to retention policy.
- [ ] Update Ubuntu packages regularly.
- [ ] Renew SSL automatically and test renewal.
- [ ] Review inventory adjustment and expense approval audit logs.
- [ ] Check disk usage for uploads, logs, and backups.
